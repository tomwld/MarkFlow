# Markdown Syntax Guide

## Headers
# H1
## H2
### H3
#### H4
##### H5
###### H6

## Emphasis
*Italic* or _Italic_
**Bold** or __Bold__
***Bold and Italic*** or ___Bold and Italic___
~~Strikethrough~~

## Lists

### Unordered
* Item 1
* Item 2
  * Sub Item 2.1
  * Sub Item 2.2
- Item 3
+ Item 4

### Ordered
1. First Item
2. Second Item
3. Third Item

### Task List
- [x] Completed task
- [ ] Incomplete task

## Links and Images
[Link Text](https://www.example.com)
![Image Alt Text](https://via.placeholder.com/150)

## Code

Inline `code`

```javascript
// Code Block
function hello() {
  console.log("Hello World");
}
```

## Blockquote
> This is a blockquote.
>
> > Nested blockquote.

## Horizontal Rule
---
***
___
